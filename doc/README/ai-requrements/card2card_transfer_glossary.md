**Дата:** 24.04.2026

## Бизнес-термины

### Карта-источник
**Категория:** Бизнес  
**Синонимы:** Карта списания, карта отправителя  
**Определение:** Банковская карта, с которой списываются денежные средства при осуществлении перевода.  
**Контекст:** Используется во всех сценариях перевода (FR-01, FR-02, FR-03).  
**Связанные термины:** Карта-назначение, Отправитель  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md)

### Карта-назначение
**Категория:** Бизнес  
**Синонимы:** Карта зачисления, карта получателя  
**Определение:** Банковская карта, на которую зачисляются денежные средства при осуществлении перевода.  
**Контекст:** Используется во всех сценариях перевода.  
**Связанные термины:** Карта-источник, Получатель  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md)

### Me-to-Me (перевод между своими картами)
**Категория:** Бизнес  
**Синонимы:** Перевод между собственными картами  
**Сокращения:** M2M  
**Определение:** Операция перевода денежных средств между картами, принадлежащими одному клиенту банка.  
**Контекст:** Бесплатный тариф, мгновенное зачисление (FR-02).  
**Связанные термины:** On-us, Карта-источник, Карта-назначение  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#fr-02-перевод-между-своими-картами-me-to-me)

### Отправитель (Sender)
**Категория:** Роль  
**Синонимы:** Плательщик, инициатор перевода  
**Определение:** Физическое или юридическое лицо, инициирующее перевод денежных средств с карты на карту.  
**Контекст:** Основной актор во всех Use Case (UC-01 — UC-06).  
**Связанные термины:** Получатель, Карта-источник  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#32-пользовательские-роли)

### Получатель (Recipient)
**Категория:** Роль  
**Синонимы:** Бенефициар, владелец карты получателя  
**Определение:** Физическое лицо, на карту которого зачисляются денежные средства.  
**Контекст:** Вторичный актор, может быть клиентом любого банка.  
**Связанные термины:** Отправитель, Карта-назначение  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#32-пользовательские-роли)

### Комиссия за перевод
**Категория:** Бизнес  
**Синонимы:** Тариф, плата за перевод  
**Определение:** Плата, взимаемая банком за осуществление перевода. Не взимается для me-to-me переводов.  
**Контекст:** Настраивается через административную панель (Раздел 7 BRD).  
**Связанные термины:** Тарифная политика  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#7-тарифная-политика)

### Лимиты переводов
**Категория:** Бизнес  
**Определение:** Ограничения на сумму и количество переводов, устанавливаемые для клиента. Включают: лимит на один перевод, суточный лимит, месячный лимит, лимит по валютам.  
**Контекст:** Настраиваются через административную панель (Раздел 8 BRD).  
**Связанные термины:** Валютный контроль  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#8-лимиты)

## Технические термины

### PAN (Primary Account Number)
**Категория:** Технический  
**Синонимы:** Номер карты  
**Определение:** Уникальный номер банковской карты, состоящий из 16-19 цифр. Используется для идентификации карты при осуществлении перевода.  
**Контекст:** Валидируется алгоритмом Луна, маскируется при отображении (FR-01).  
**Связанные термины:** Алгоритм Луна, Платежная система  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#fr-01-перевод-по-номеру-карты)

### On-us
**Категория:** Технический  
**Определение:** Операция перевода, при которой карты отправителя и получателя выпущены одним банком. Обрабатывается внутри банка без обращения к внешним платежным системам.  
**Контекст:** Мгновенное зачисление, бесплатный тариф (FR-02).  
**Связанные термины:** Off-us, Me-to-Me  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#fr-02-перевод-между-своими-картами-me-to-me)

### Off-us
**Категория:** Технический  
**Определение:** Операция перевода, при которой карты отправителя и получателя выпущены разными банками. Требует маршрутизации через платежную систему.  
**Контекст:** Время обработки ≤ 30 секунд, взимается комиссия.  
**Связанные термины:** On-us, Платежный шлюз  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#51-производительность)

### Платежный шлюз
**Категория:** Технический  
**Синонимы:** Payment Gateway  
**Определение:** Система, обеспечивающая маршрутизацию и обработку транзакций между банком и платежными системами (Visa, Mastercard, МИР).  
**Контекст:** Интеграция с внешними системами (Раздел 10 BRD).  
**Связанные термины:** Платежная система, Off-us  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#102-внешние-системы)

### Антифрод-система
**Категория:** Технический  
**Синонимы:** Fraud Monitoring System  
**Определение:** Система мониторинга и предотвращения мошеннических операций, анализирующая транзакции в реальном времени.  
**Контекст:** Обеспечение безопасности переводов (Раздел 5.2 BRD).  
**Связанные термины:** Двухфакторная аутентификация  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#52-безопасность)

### Алгоритм Луна (Luhn algorithm)
**Категория:** Технический  
**Определение:** Алгоритм проверки контрольной суммы номера банковской карты, используемый для валидации PAN.  
**Контекст:** Применяется при вводе номера карты получателя (FR-01).  
**Связанные термины:** PAN  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#fr-01-перевод-по-номеру-карты)

### Core Banking System (ABS)
**Категория:** Технический  
**Синонимы:** Автоматизированная банковская система (АБС)  
**Определение:** Центральная система банка, обеспечивающая ведение счетов, проводки и управление остатками.  
**Контекст:** Интеграция для выполнения проводок по счетам (Раздел 10.1 BRD).  
**Связанные термины:** On-us  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#101-внутренние-системы)

## Термины данных

### Транзакция
**Категория:** Данные  
**Синонимы:** Операция, перевод  
**Определение:** Запись о переводе денежных средств, содержащая: ID транзакции, дату, сумму, валюту, комиссию, статус, карты отправителя и получателя.  
**Контекст:** Сохраняется в истории операций (FR-04).  
**Связанные термины:** История операций  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#fr-04-история-операций)

### Шаблон перевода
**Категория:** Данные  
**Определение:** Сохраненный набор реквизитов получателя (название, маскированный номер карты, банк получателя) для быстрого повторного перевода.  
**Контекст:** Не более 50 шаблонов на пользователя (FR-05).  
**Связанные термины:** PAN  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#fr-05-шаблоны-переводов)

## API термины

### REST API
**Категория:** API  
**Определение:** Архитектурный стиль взаимодействия компонентов системы через HTTP-запросы. Используется для интеграции с внутренними и внешними системами.  
**Контекст:** Взаимодействие между мобильным приложением, интернет-банком и backend-сервисами.  
**Связанные термины:** Платежный шлюз  
**Источник:** [`card2card_transfer_brd.md`](card2card_transfer_brd.md#10-интеграции)

## Акронимы и сокращения

| Сокращение | Полное название | Определение |
|------------|----------------|-------------|
| **PAN** | Primary Account Number | Номер банковской карты |
| **M2M** | Me-to-Me | Перевод между своими картами |
| **MVP** | Minimum Viable Product | Минимально жизнеспособный продукт |
| **AML** | Anti-Money Laundering | Противодействие отмыванию денег |
| **ПОД/ФТ** | Противодействие отмыванию доходов и финансированию терроризма | Российский аналог AML |
| **СБП** | Система быстрых платежей | Система Банка России для мгновенных переводов |
| **OTP** | One-Time Password | Одноразовый пароль для подтверждения операций |
| **PCI DSS** | Payment Card Industry Data Security Standard | Стандарт безопасности данных индустрии платежных карт |
| **NPS** | Net Promoter Score | Индекс потребительской лояльности |
| **RPO** | Recovery Point Objective | Целевая точка восстановления данных |
| **RTO** | Recovery Time Objective | Целевое время восстановления системы |
| **ABS** | Automated Banking System | Автоматизированная банковская система |
